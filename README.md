Project Title

SoulSync — Mood NFT Journal

Project Description

SoulSync is a simple Web3 application that allows users to record their daily emotions in the form of “Mood Capsules.”
Each saved mood is automatically transformed into a private or semi-public NFT featuring unique abstract visuals generated from the user’s emotions of the day.

Examples
Feeling sad today → a blue rainy-themed NFT
Feeling energetic → a bright colorful NFT with fast animations
Feeling overwhelmed or overthinking → an abstract glitch-pattern NFT

These NFTs are not designed for expensive trading or speculation, but rather as an emotional on-chain digital diary with personal meaning.

What Makes It Unique
The NFTs are not ordinary uploaded artworks
NFT visuals are generated from a combination of the user’s mood and written thoughts
Every user has their own emotional timeline stored on the blockchain

Project Vision
To make blockchain technology feel more personal and emotional — not just about crypto, finance, and trading.

Feature List
- Wallet login (MetaMask)
- Daily mood selection
- Short personal notes/journal entries
- Automatic abstract visual generation
- Mint mood entries as NFTs
- Emotional journey timeline
- Monthly mood statistics
- “Memory Replay” feature to revisit moods from previous years
- Private/Public capsule options
- Share capsules with friends using unique links
