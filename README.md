# SoulSync — Mood NFT Journal

**SoulSync** — Emotional Web3 Journal Powered by NFTs

## Project Description

SoulSync is a decentralized Web3 application that allows users to capture and preserve their daily emotions through blockchain technology. Instead of simply writing notes in a traditional journal, users create “Mood Capsules” — emotional records that are automatically transformed into unique NFTs with abstract visuals generated from the user’s mood and personal reflections.

Each mood entry becomes a deeply personal digital artifact stored on-chain, creating an emotional timeline that users can revisit over time. The generated NFT visuals are dynamic and emotion-driven, making every capsule unique and meaningful rather than just another collectible image.

Examples:
- Feeling sad → a calm blue rainy-themed NFT
- Feeling energetic → a vibrant colorful NFT with fast-moving abstract patterns
- Feeling overwhelmed → a glitch-inspired chaotic abstract NFT

Unlike conventional NFT projects focused on speculation and trading, SoulSync focuses on emotional expression, memory preservation, and personal storytelling through blockchain technology.
